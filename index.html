CHALLENGE 1: To-Do List

This solo project spanned two weeks. I built a small HTML frame, styled it with CSS, and used JavaScript modules to create a mobile-friendly web-based app that uses LocalStorage to allow the user to save, sort, check off, and delete items on a simple to-do list. Things I'm proud of:

  • Modules and modular code are new to me, but I think I'm improving at streamlining the code and making it more portable (though still far from perfect)
  • Clicking back and forth was bugging me, so I learned how to bind an event to the "enter" key, which actually turned out to be really helpful when testing on my phone as well.
  • I designed a very simple check for user input that keeps the list from saving an empty field, subtly lighting up the text box and refocusing the cursor in the box if the button is clicked without entering a task first.
  • I think it's pretty. :)
