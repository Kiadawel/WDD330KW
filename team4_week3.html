Array Cardio 1

Array.filter()

In a given array of inventors, this filters and lists the ones born in the 1500s.

Array.map()

Takes the same array of inventors, and lists all of them by first and last name.

Array.sort()

Sorts the inventors by birth year.

Array.reduce()

Figures out how old each inventor was, and adds up all of their ages.

Array.sort(v2)

Sorts the inventors by their ages.

Sort People

Sorts a list of people by their last names.

Count Data

Searches through an array and counts up the instances of each particular bit of data.

Array Cardio 2

Array.some() and Array.every()

Checks an array of people to see if some people are over 19, then if every person is over 19.

Array.find()

This searches an array for a given ID and shows you the comment that goes with it.

Array.findIndex()

Finds a certain entry in the comments array and deletes it. (Results visible in the console only.)
